## Цель работы 
разработать структурированный информационно‑справочный раздел «Я и цифровой мир», охватывающий ключевые аспекты взаимодействия человека с цифровой средой: зависимость от гаджетов, игровую культуру, информационную гигиену, кибербезопасность, баланс реальной и виртуальной жизни, а также уход за техникой.
В рамках проекта необходимо:

построить онтологию предметной области с визуализацией связей между темами;

собрать и проанализировать данные из Wikidata с помощью SPARQL-запросов;

подготовить не менее 35 коротких статей в формате Markdown, раскрывающих каждое понятие;

организовать файловую структуру репозитория по заданному шаблону (WORK/ и WEB/);

автоматизировать генерацию контента с использованием LLM (DeepSeek) и предоставить отчёт о процессе работы.
