# Отчёт по подтеме «Мои игры»

**Ответственный:** Мария

## Связи внутри подтемы
- Игры как хобби → отличие от зависимости
- Киберспорт → тренировки, команды, турниры
- Игровые сообщества → дружба и токсичность
- Донаты → психология микроплатежей
- Замена игр → альтернативные увлечения

## SPARQL-запрос
Запрос на количество участников крупнейших игровых сообществ.

## Результаты выгрузки
Топ-2: Discord Gaming (150 млн), Steam Community (120 млн).

## Процесс
Нашли свойство `P1082` (количество участников) для сущностей типа `Q8685` (игровое сообщество).

## Ощущения
Цифры огромные — игровые сообщества сопоставимы с населением стран.
